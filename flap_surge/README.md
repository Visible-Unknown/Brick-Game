# 🐦 Neo Flap-Surge: Voltage Run
### *High-Fidelity One-Button Reaction Runner*

**Neo Flap-Surge** is a brutal, adrenaline-pumping survival game. Tap into the frequency, navigate a bird-drone through deadly voltage gates, and survive the accelerating neon city.

---

## 🕹️ Gameplay Features
- **🌐 Full-Screen Immersion**: Multi-layered parallax neon city backgrounds.
- **✨ Professional UI**: Stroke-font titles, transparent HUD, and "Circuit Broken" game-over screens.
- **⚡ Pulsing Obstacles**: Gates pulse with neon energy, creating a high-contrast challenge.
- **⚙️ Escalating Difficulty**: The world scroll speed and pipe frequency increase as your score climbs.
- **🕊️ Physics-Based Flight**: Tuned gravity and lift mechanics for satisfying, precise control.
- **🎵 Pro Audio Engine**: Driving industrial synth BGM and tactical SFX.

---

## 🎮 Controls
| Action | Key |
| :--- | :--- |
| **Flap** | `SPACE` |
| **Start / Restart** | `SPACE` (on menus) |
| **Pause / Resume** | `P` |
| **Quit** | `ESC` |

---

## 🛠️ Build Instructions
```bash
g++ flap_surge.cpp -o flap_surge.exe -lfreeglut -lopengl32 -lglu32 -lwinmm
```
