# ☄️ Neo Asteroids: Void Drift
### *High-Fidelity Vector Space Survival*

**Neo Asteroids** is a minimalist yet intense space brawler. Pilot a high-performance vector-tri ship through a dense asteroid field, splitting rocks with precision fire and surviving aggressive UFO incursions.

---

## 🕹️ Gameplay Features
- **🌐 Full-Screen Immersion**: Pure vector-wireframe neon rendering.
- **✨ Professional UI**: Stroke-font titles, lives counter, and score pulse.
- **🎮 Physics-Based Controls**: Realistic rotation, thrust, and drift mechanics.
- **🧱 Rock Splitting**: Large asteroids split into medium ones, then smaller fragments.
- **🛸 UFO AI**: Hostile scout ships warp in periodically to target and engage the player.
- **🌌 Screen-Wrap Logic**: Seamless wraparound physics for a continuous battlefield.
- **🎵 Pro Audio Engine**: High-energy synth BGM and crisp vector-explosive SFX.

---

## 🎮 Controls
| Action | Key |
| :--- | :--- |
| **Thrust** | Up Arrow |
| **Rotate** | Left / Right Arrow |
| **Fire** | `SPACE` |
| **Start / Restart** | `SPACE` (on menus) |
| **Pause / Resume** | `P` |
| **Quit** | `ESC` |

## 🛠️ Build Instructions
```bash
g++ asteroids.cpp -o asteroids.exe -lfreeglut -lopengl32 -lglu32 -lwinmm
```
